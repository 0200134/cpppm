ğŸ”§ Quick Start

# Compile cpppm
c++ -std=c++17 -O2 -o cpppm tools/cpppm.cpp

# Initialize
./cpppm init

# Add dependencies
./cpppm add fmt https://github.com/fmtlib/fmt.git tag=10.2.1
./cpppm add catch2 https://github.com/catchorg/Catch2.git tag=v3.5.4

# Install + Build
./cpppm install
cmake -B build -S .
cmake --build build
